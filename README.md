# ImageClusterization

## Requirements
*Visual C++ 11*<br>
*OpenCV*<br>
*Boost*

## Usage
build project and run with parametrs:

`clusterization.exe -f file1 file2...`<br>
for use on standalone files.

Or:<br>
`clusterization.exe -d path`<br>
where path is a path to direcroty with images(using recurcive search).
